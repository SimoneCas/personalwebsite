Elenco delle principali piattaforme web testate per le seguenti tipologie di servizi:

  • TopHost: Il canone annuo per un domino e' 5.99 EUR, il piu' economico che abbia mai visto.
  • Gandi.net:Conveniente se si ha bisogno dei certificati SSL, incluso nel dominio di un anno da 12.99 EUR
Server:
  • Server4You: Virtual server economico
  • ShellRent:Servizio piu' stabile ma un pochino piu' costoso.

  • www.hetzner.de/
  • www.aruba.it/
Dominio + Hosting linux:
  • DnsHosting: Usato maggiormente per siti basati su CMS tipo Joomla e Wordpress, che in genere include anche i DBMS
Servizi gratis:
  • www.heroku.com: Configurazione base, gia' tutto installato, permettendo così di concentrasi sul codice da sviluppare. Deploy con Git
  • aws.amazon.com

PNGGauntlet

Open Tool utile per comprimere immagini senza perdita di dati.
Permette di convertire JPG, GIF, TIFF e BMP in PNG.
Ottimizza la grandezza delle immagini tramite 7 diversi livelli di compressione, diminuendo così il tempo di caricamento delle pagine html che le utilizzano.

http://pnggauntlet.com/
