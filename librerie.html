Ribbon

Ribbon is a client side IPC library that is battle-tested in cloud. It provides the following features

  • Load balancing
  • Fault tolerance
  • Multiple protocol (HTTP, TCP, UDP) support in an asynchronous and reactive model
  • Caching and batching

Project Lombok

Libreria Java che permette di autogenerare codice standard nelle classi di modello: permette di autogenerare getter e setter, costruttori, metodi toString e hashCode, autogenerando anche funzionalità di Builder.

Tool di progetto

Ottobre 11, 2019

Elenco di tutti gli strumenti di analisi, monitoraggio e progettazione che a mio avviso non possono e non devono mancare per una sana gestione di un progetto informatico:

  • JavaDoc e JavaDoc Maven Plugin: libreria per autogenerare a partire dai commenti nel codice la documentazione di progetto.
  • JaCoCo: libreria open source per analisi della coverage dei test unitari tramite generazione di report html
  • Jenkins: Fornisce servizi di Continuous Integration per lo sviluppo del software
  • Sonarqube: Piattaforma open-source dedicata per la continuous inspection della qualità del codice, automatizzando analisi statica del codice, individuazione di bugs, code smells e vulnerabilità di sicurezza
  • jbehave: Framework basato su Behaviour-Driven Development (BDD) utile per generare test di sistema descritti in modo testuale, facilitando così la fase di collaudo e la coverage dei requisiti rispetto il prodotto sviluppato
  • JUnit 5: Libreria utilizzata per lo sviluppo dei test unitari. Adottare la nuova versione 5. Guida alle differenze tra JUnit 4 e JUnit 5
