Ultimo Articolo

Se utilizzando Coherence si decide di serializzare gli oggetti tramite Portable Object Format (POF) Serialization bisognerà dichiarare all'interno del file pof-config.xml il mapping tra la classe da serializzare ed il relativo serializzatore.
Se la classe da serializzare è una Inner Class dovremo utilizzare la seguente sintassi per indicare la classe oggetto di serializzazione.
Nell'esempio successivo viene utilizzato il carattere '$' per indicare che BetType è una InnerClass dichiarata all'interno della classe Bet. 

<user-type>
    <type-id>1001</type-id>
    <class-name>it.sample.model.Bet$BetType</class-name>
    <serializer>
       <class-name>com.tangosol.io.pof.EnumPofSerializer</class-name>
    </serializer>
</user-type>
